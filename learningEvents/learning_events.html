Keydown(press) and keyup(release)

The keyup event triggers upon releasing a key,
while the keydown event triggers upon pressing a key down.

Keypress(press and hold)

The keypress event in JavaScript occurs when a key on the keyboard
is pressed down and produces a character value. It specifically
represents pressing a key that results in a character input.

1 Submit event

Activated when a user submits the form, either by clicking
a submit button or programmatically using JavaScript.
This event enables developers to handle form data, perform
validation, or prevent default submission behavior.

Change event

Used when the value of an input element within the form changes.
It applies to various form elements such as text inputs, checkboxes,
radio buttons, and dropdowns. This event enables real-time validation,
updates, or actions based on user input changes.

Focus event

Focus event triggers element attention (for example: clicks, tabs).
These events are useful for providing visual cues, validation messages,
or triggering actions when users interact with form elements.

Used when the entire document and its dependent resources
